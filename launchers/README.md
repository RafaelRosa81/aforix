# Aforix launchers

This directory contains small helper scripts for starting Aforix from Windows or Linux without typing long commands.

Launchers do not implement processing logic. They only:

- activate the configured environment;
- move to the Aforix repository;
- run existing `aforix` commands or batch YAML files;
- keep the terminal visible so users can read the result.
